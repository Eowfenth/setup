### Troubleshooting

#### 1. Permission denied

Change permissions of the file you want to run.

'''bash
chmod 775 filename
'''
Solution explained by: [sandyd at ubuntuforums](https://ubuntuforums.org/showthread.php?t=2260192&s=48ac9c479dafecd16a43811c19735f2c&p=13204468#post13204468)
